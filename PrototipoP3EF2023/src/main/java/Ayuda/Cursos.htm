Mantenimiento de Cursos

En el mantenimiento de Cursos tiene como objeto registrar un nuevo Curso, sucesivo a eso realizar las siguientes opciones:

Modificar

Eliminar

Buscar

Ayuda

Reporte

Para ingresar un nuevo registro debemos dirigirnos al menú de catalogos en el modulo General y seleccionar la opción de "Mantenimientos", seguidamente la opción de "Mantenimiento Cursos

Despues podremos observar el formulario a llenar para agregar una nueva aplicación.
